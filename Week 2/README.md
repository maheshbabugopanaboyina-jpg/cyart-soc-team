# Week 2 - SOC Training

## Overview
This repository contains all tasks performed as part of Week 2 SOC training, covering alert management, incident response, triage, and evidence preservation.

---

## Tasks Completed

### 1. Alert Classification
- Created alert classification table
- Mapped alerts to MITRE ATT&CK techniques

### 2. CVSS Prioritization
- Assigned priorities based on CVSS scores
- Understood severity levels (Critical, High, Medium, Low)

### 3. Incident Ticket Creation
- Created a ransomware incident ticket
- Included IOCs and response steps

### 4. Escalation Process
- Drafted escalation email to Tier 2 SOC team

### 5. Alert Triage
- Analyzed alerts and validated using threat intelligence
- Identified false positives

### 6. Evidence Preservation
- Documented evidence collection process
- Learned hashing and chain of custody

### 7. Capstone Project
- Simulated attack using Metasploit
- Detected, analyzed, and responded to incident
- Created final report and stakeholder briefing
